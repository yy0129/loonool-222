第一次用 LOONOOL，怎么开始？
建议路径：
① 登录 / 注册账号 → ② 在首页创建一个图片审核空间 → ③ 在空间内上传候选图片 → ④ 使用 Evidence & Tools 做决策 → ⑤ 选出 Final 图并导出结果。
图片审核空间（Space）是一个专门用于审核图片的协作空间。 每个空间可以包含多个任务（Task），每个任务内可以放多张候选图片、讨论消息、证据（Evidence）和 AI 分析结果，让整个决策过程可追溯、可复盘。
Task 是围绕一个具体主题的一组候选图片，例如「首页 KV」「电商主图」。 每个 Task 下可以上传 1–6 张候选图片，团队在这个任务里进行讨论、添加 Evidence，并最终选出一张 Final 图，用于真实投放或上线。
为了让讨论更聚焦，每个 Task 最多支持 6 张候选图片。 当你继续上传第 7 张时，系统会自动为你创建一个新的 Task，并把新图片放到新的 Task 中，避免一个任务里图片过多、难以做出明确结论。
Evidence 是你在决策过程中产生的所有依据，例如平台规则 PDF、竞品截图、AI 分析结果等。

Add Evidence 是上传这些依据的入口，可以上传图片、PDF / 文档，也可以添加外部链接。

Analysis Tools 是内置 AI 工具入口，例如风险扫描、风格分析、市场案例等。工具执行后的结果，可以一键保存为 Evidence。
在审核空间左侧 Sidebar 中，点击「导出结果（Export Results）」按钮，系统会收集当前空间所有已选定 Final 图的任务，将对应图片打包为 zip 下载。后续版本会逐步支持带摘要说明和元数据的导出格式。
每条 Evidence 都可以生成一个在线预览链接（Evidence Preview）。 你可以将链接分享给品牌方、法务或代理商，对方可以在不登录的情况下查看单条 Evidence 的内容；如果想查看完整证据链，则需要进入对应的 Space / Task。
