LOONOOL · 图片审核空间 · 常见问题 FAQ

我应该从哪一步开始？
建议流程：先登录 / 注册 → 在首页创建一个图片审核空间 → 在空间内上传候选图片 → 使用 Evidence & Tools 做决策 → 选出 Final 图并导出结果。
什么是图片审核空间？
图片审核空间（Space）是一个专门用于审核图片的协作空间。 每个空间可以包含多个任务（Task），每个任务内可以放多张候选图片、讨论消息、证据（Evidence）和 AI 分析结果。
什么是任务（Task）？
Task 是围绕一个具体主题的一组候选图片，例如「首页 KV」「电商主图」。 每个 Task 下可以上传 1–6 张候选图片，团队在这个任务里进行讨论、添加 Evidence，并最终选出一张 Final 图。
为什么一个任务最多只能放 6 张图片？
为了让讨论更聚焦，每个 Task 最多支持 6 张候选图片。当你继续上传第 7 张时，系统会自动为你创建一个新的 Task，并把新图片放到新的 Task 中，避免一个任务里图片过多、难以决策。
什么是 Evidence / Add Evidence / Analysis Tools？
Evidence 是你在决策过程中产生的所有依据，例如平台规则 PDF、竞品截图、AI 分析结果等。

Add Evidence 是上传这些依据的入口，可以上传图片、PDF / 文档，也可以添加外部链接。

Analysis Tools 是内置 AI 工具入口，例如风险扫描、风格分析、市场案例等。工具执行后的结果，可以一键保存为 Evidence。
如何导出结果？
在审核空间左侧 Sidebar 中，点击「导出结果（Export Results）」按钮，系统会收集当前空间所有已选定 Final 图的任务，将对应图片打包为 zip 下载。本功能在后续版本中会支持更多导出格式。
如何分享证据给外部？
每条 Evidence 都可以生成一个在线预览链接（Evidence Preview）。 你可以将链接分享给品牌方、法务或代理商，对方可以在不登录的情况下查看单条 Evidence 的内容；如果想查看完整证据链，则需要进入对应的 Space / Task。
