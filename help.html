帮助中心 / 常见问题

简要说明 LOONOOL V1 的核心概念与使用方式。

什么是图片审核空间？
图片审核空间是一个团队协作环境，在图片对外使用前，团队可以在这里做多图对比、讨论、使用 AI 工具，并形成可追溯的证据链。
什么是任务（Task）？
Task 代表一次具体的视觉决策，例如「首页主图选图」或「某次包装图审核」。一个空间中可以有多个 Task，每个 Task 下有自己的图片、讨论和证据。
为什么一个任务最多只能放 6 张图片？
每个 Task 限制 1–6 张图片，是为了让每次审核聚焦于少量「方向」，避免一次性塞入大量图片导致讨论发散、决策困难。
什么是 Evidence / Add Evidence / Analysis Tools？
Evidence 是支撑你做出决策的证据，比如：AI 分析报告、平台规则 PDF、截图或人工整理说明。
Add Evidence 是把这些材料上传到系统的入口。
Analysis Tools 是一组 AI 工具，用来分析图片风险、风格与市场表现，并以 Evidence 的形式保存结果。
